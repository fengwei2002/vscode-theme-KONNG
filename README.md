# KONNG

知乎同步教程：[如何自定义一个vscode主题插件](https://zhuanlan.zhihu.com/p/157288300)

基于 bluloco Dark Theme 3.1.1 进行优化后的一个主题

- 终端颜色更加自然，原版的命令行体验不是很好
- 统一将晃眼睛的蓝色改为紫罗兰色 #9400D3
- 其他小细节的个人优化
